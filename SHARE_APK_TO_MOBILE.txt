STEP 1: PREPARE YOUR ANDROID PHONE (1-2 MINUTES)
═════════════════════════════════════════════════════════════════════════════

Goal: Enable USB Debugging

ACTION:
───────

1. Open Settings app on your phone

2. Scroll down and find "About Phone" (might be at bottom)

3. Look for "Build Number" field (usually near bottom)

4. Tap "Build Number" repeatedly - TAP 7 TIMES QUICKLY
   You'll see a countdown: 6, 5, 4, 3, 2, 1
   Then message: "You are now a developer" ✓

5. Go back to Settings main screen

6. Scroll down - "Developer Options" should now appear

7. Tap "Developer Options"

8. Find "USB Debugging" toggle

9. Turn ON USB Debugging (should be blue/green)

10. A popup appears asking "Allow USB debugging?"

11. Tap "OK" or "Always Allow from this computer"

✓ USB DEBUGGING IS NOW ENABLED

❌ TROUBLESHOOTING DURING INSTALLATION
═════════════════════════════════════════════════════════════════════════════

Problem: "USB Debugging not showing"
────────────────────────────────────
Solution:
   1. Go to About Phone
   2. Find Build Number
   3. Tap Build Number 7 times
   4. Return to Settings
   5. Developer Options should appear

Problem: "Device not detected"
──────────────────────────────
Solution:
   1. Disconnect USB cable
   2. Wait 5 seconds
   3. Reconnect USB cable
   4. Look for permission prompt on phone
   5. Tap "Allow"

Problem: "Installation command not recognized"
──────────────────────────────────────────────
Solution:
   1. Make sure you're in correct directory:
      cd C:\Users\Admin\Downloads\gokul-stores
   2. Try using install-apk.bat instead (OPTION 1)

Problem: "App doesn't appear in app drawer"
────────────────────────────────────────────
Solution:
   1. Restart phone
   2. Check if installation completed successfully
   3. Try reinstalling: adb uninstall com.gokul.stores
   4. Then: adb install -r dist-apk/gokul-stores-debug.apk

Problem: "App crashes on startup"
─────────────────────────────────
Solution:
   1. Ensure phone has internet connection
   2. Clear app cache: Settings → Apps → Gokul Stores → Clear Cache
   3. Uninstall and reinstall app

Problem: "Firebase connection error"
────────────────────────────────────
Solution:
   1. Check phone has internet connection
   2. Try again
   3. If still failing, restart phone
   4. Reinstall app
