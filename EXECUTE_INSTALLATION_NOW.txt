RIGHT NOW - PREPARE YOUR PHONE (1 MINUTE)
═════════════════════════════════════════════════════════════════════════════

YOUR ACTION:
────────────

1. PICK UP YOUR ANDROID PHONE

2. OPEN: Settings app

3. SCROLL DOWN: Find "About Phone"

4. FIND: "Build Number" (usually at bottom)

5. TAP IT: 7 TIMES QUICKLY
   (Count the taps: 1, 2, 3, 4, 5, 6, 7)

   → You'll see countdown and message about "Developer Mode"

6. GO BACK: To Settings main screen

7. TAP: "Developer Options" (now visible)

8. SCROLL: Find "USB Debugging"

9. TOGGLE: Turn it ON (blue/green)

10. TAP: "OK" when prompted

✅ DONE! Your phone is now in Developer Mode with USB Debugging enabled

NEXT - CONNECT PHONE TO COMPUTER (30 SECONDS)
═════════════════════════════════════════════════════════════════════════════

YOUR ACTION:
────────────

1. GET YOUR USB CABLE

2. PLUG INTO PHONE: Connect USB cable to phone's charging port

3. PLUG INTO COMPUTER: Connect other end to computer's USB port

4. WAIT: A few seconds...

5. LOOK AT PHONE SCREEN: You'll see permission prompt
   "Allow USB debugging from this computer?"

6. TAP: "ALLOW" or "Always allow from this computer"

✅ DONE! Your phone is now connected to your computer

──────────────────────────────────────────

4. PRESS: Enter

5. WAIT: Installation happens

❌ PROBLEMS? QUICK FIXES

"USB Debugging not visible"
   → Go to About Phone
   → Tap Build Number 7 times
   → Restart Settings
   → Try again

"Device not detected"
   → Try different USB cable
   → Try different USB port
   → Restart phone
   → Restart computer

"install-apk.bat not found"
   → Make sure you're in: C:\Users\Admin\Downloads\gokul-stores
   → Check if File Explorer shows all files
   → Use Method 2 (manual ADB) instead

"Installation fails"
   → Run: adb uninstall com.gokul.stores
   → Then try installing again

"App crashes on startup"
   → Clear phone cache: Settings → Apps → Gokul Stores → Clear Cache
   → Reinstall app
   → Make sure phone has internet
