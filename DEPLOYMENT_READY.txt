🚀 QUICK INSTALLATION (3 STEPS)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: PREPARE YOUR PHONE
──────────────────────────
1. Go to Settings → Developer Options
   (If Developer Options not visible: Settings → About Phone →
    Tap "Build Number" 7 times)
2. Enable "USB Debugging"
3. Connect phone to computer with USB cable
4. Accept USB debugging prompt on phone

STEP 2: INSTALL APK
───────────────────
Option A (Automatic) - RECOMMENDED:
   1. Navigate to: C:\Users\Admin\Downloads\gokul-stores
   2. Double-click: install-apk.bat
   3. Wait for success message

Option B (Manual):
   1. Open Command Prompt
   2. Run: adb install -r dist-apk/gokul-stores-debug.apk

Option C (Android Studio):
   1. Open android/ folder in Android Studio
   2. Click Run → Run 'app'

STEP 3: LAUNCH APP
──────────────────
1. Find "Gokul Stores" in your app drawer
2. Tap to open
3. Sign in or create account
4. Start shopping! 🛍️

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

"USB Debugging not working?"
  → Enable Developer Options first (tap Build Number 7 times)
  → Restart phone and computer
  → Try different USB cable
  → Run: adb devices (check if device is listed)

"Installation fails?"
  → Uninstall existing version first
  → Run: adb uninstall com.gokul.stores
  → Then reinstall: adb install -r dist-apk/gokul-stores-debug.apk

"App crashes on startup?"
  → Ensure internet connection
  → Clear app cache: Settings → Apps → Gokul Stores → Storage → Clear Cache
  → Reinstall the app

"Can't find USB Debugging option?"
  → Go to Settings → About Phone
  → Tap on Build Number 7 times
  → Developer Options should now appear in Settings
