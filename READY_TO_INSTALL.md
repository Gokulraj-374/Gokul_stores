## 🚀 START HERE

### The Easiest Way (Recommended) ⭐

**5 minutes total**

1. **Prepare Phone** (1 minute)
   - Settings → About Phone → Tap "Build Number" 7 times
   - Settings → Developer Options → Enable USB Debugging

2. **Connect Phone** (30 seconds)
   - Plug USB cable into phone and computer
   - Accept permission prompt on phone

3. **Install App** (30 seconds)
   - Navigate to: `C:\Users\Admin\Downloads\gokul-stores`
   - **Double-click: `install-apk.bat`**
   - Wait for "Success" message
   - **Done!** ✅

4. **Launch App** (1 minute)
   - Open app drawer on phone
   - Find "Gokul Stores"
   - Tap to launch
   - Sign in or create account

## ❌ QUICK TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| Developer Options not visible | Tap Build Number 7 times in About Phone |
| USB Debugging not working | Restart phone, try different cable/port |
| Installation fails | Run: `adb uninstall com.gokul.stores` then reinstall |
| App crashes | Clear cache: Settings → Apps → Clear Cache |
| Device not detected | Update USB drivers, restart computer |
