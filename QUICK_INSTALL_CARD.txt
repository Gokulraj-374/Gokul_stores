⚡ 5-MINUTE INSTALLATION
═════════════════════════════════════════════════════════════════════════════

MINUTE 1: ENABLE USB DEBUGGING ON PHONE
────────────────────────────────────────
1. Settings → About Phone
2. Tap Build Number 7 times
3. Settings → Developer Options
4. Enable USB Debugging
5. Allow USB debugging permission

MINUTE 2-3: CONNECT PHONE
──────────────────────────
1. Connect phone to computer with USB cable
2. Accept debugging permission prompt

MINUTE 4-5: INSTALL APK
─────────────────────────
Method 1 (EASIEST):
   Double-click: install-apk.bat
   Wait for "Success" message
   Done!

Method 2 (MANUAL):
   Run in PowerShell:
   adb install -r dist-apk/gokul-stores-debug.apk

🔧 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Can't find Developer Options?
   → Go to About Phone, tap Build Number 7 times

Device not detected?
   → Try different USB port or cable
   → Restart phone and computer

Installation failed?
   → Run: adb uninstall com.gokul.stores
   → Then reinstall: adb install -r dist-apk/gokul-stores-debug.apk

App crashes?
   → Clear cache: Settings → Apps → Gokul Stores → Storage → Clear Cache
   → Reinstall the app
