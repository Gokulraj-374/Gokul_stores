🚀 INSTALLATION GUIDE
═══════════════════════════════════════════════════════════════════════════════

STEP 1: PREPARE YOUR PHONE
──────────────────────────

1. Go to Settings → Developer Options
   (If not visible: Settings → About Phone → Tap "Build Number" 7 times)

2. Enable "USB Debugging"

3. Accept USB debugging permission on your phone when asked

4. Connect phone to computer with USB cable

STEP 2: INSTALL APK
───────────────────

OPTION A (Automatic - RECOMMENDED):

  1. Open File Explorer
  2. Navigate to: C:\Users\Admin\Downloads\gokul-stores
  3. Double-click: install-apk.bat
  4. Wait for "Installation successful" message

  Installation will complete in ~30 seconds

OPTION B (Manual Command):

  1. Open PowerShell
  2. Run command:
     adb install -r dist-apk/gokul-stores-debug.apk
  3. Wait for installation to complete

OPTION C (Android Studio):

  1. Open Android Studio
  2. File → Open → Select android/ folder
  3. Connect your device
  4. Click Run (green play button)
  5. Select your device
  6. App will install and launch

STEP 3: LAUNCH APP
──────────────────

1. Look for "Gokul Stores" in your app drawer
2. Tap the icon to launch the app
3. Sign in or create a new account
4. Start shopping! 🛍️

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "USB Debugging option not visible"
Solution:
  1. Go to Settings → About Phone
  2. Find "Build Number" field
  3. Tap it 7 times rapidly
  4. Developer Options will appear in Settings
  5. Enable USB Debugging

Problem: "Installation fails"
Solution:
  1. Uninstall existing version: adb uninstall com.gokul.stores
  2. Try again: adb install -r dist-apk/gokul-stores-debug.apk
  3. Or restart your phone and try again

Problem: "App crashes on startup"
Solution:
  1. Ensure phone has internet connection
  2. Clear app cache: Settings → Apps → Gokul Stores → Storage → Clear Cache
  3. Reinstall the app
  4. Check if Google Play Services is installed

Problem: "Can't find ADB"
Solution:
  1. Install Android SDK Platform Tools
  2. Add to PATH environment variable
  3. Or use Android Studio's built-in terminal
  4. Or double-click install-apk.bat instead

Problem: "Phone not detected by ADB"
Solution:
  1. Try different USB cable
  2. Try different USB port
  3. Update USB drivers
  4. Restart phone and computer
  5. Re-accept USB debugging permission
